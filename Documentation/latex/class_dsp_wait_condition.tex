\subsection{Detailed Description}
Cross-\/platform, object-\/oriented conditional wait. 

A wait condition works like an indefinite sleep. When a thread calls the Wait() function it is put to sleep until it is woken by the Wake\-All() function of the same \doxyref{Dsp\-Wait\-Condition}{p.}{class_dsp_wait_condition} object. This is used to synchronize actions between threads.
