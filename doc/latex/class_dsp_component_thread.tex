\subsection{Detailed Description}
Thread class for ticking and reseting a single component. 

A \doxyref{Dsp\-Component\-Thread}{p.}{class_dsp_component_thread} is responsible for ticking and reseting a single component continuously in a separate free-\/running thread. On construction, a reference to the component must be provided for the \doxyref{Dsp\-Thread}{p.}{class_dsp_thread}'s \-\_\-\-Run() method to use. Once Start() has been called, the thread will begin repeatedly executing the \-\_\-\-Run() method. On each thread iteration, \doxyref{Dsp\-Component\-Thread}{p.}{class_dsp_component_thread} simply calls the reference component's Tick() and Reset() methods. The Pause() method causes \doxyref{Dsp\-Component\-Thread}{p.}{class_dsp_component_thread} to wait until instructed to Resume() again.
