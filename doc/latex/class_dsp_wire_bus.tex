\subsection{Detailed Description}
\doxyref{Dsp\-Wire}{p.}{struct_dsp_wire} container. 

A \doxyref{Dsp\-Wire\-Bus}{p.}{class_dsp_wire_bus} contains Dsp\-Wires (see \doxyref{Dsp\-Wire}{p.}{struct_dsp_wire}). Each component contains an input wire bus. Via the Tick() method, a \doxyref{Dsp\-Component}{p.}{class_dsp_component} uses it's input wire bus to retrieve it's input signals from incoming linked components, as mapped out in each \doxyref{Dsp\-Wire}{p.}{struct_dsp_wire}. The \doxyref{Dsp\-Circuit}{p.}{class_dsp_circuit} class has an additional 2 wire buses use to connect the circuit's I\-O signals to and from it's internal components.
