\subsection{Detailed Description}
Dynamically typed variable. 

\doxyref{Dsp\-Run\-Type}{p.}{class_dsp_run_type} holds a variable that can be dynamically typed at run-\/time (hence the name). The \doxyref{Dsp\-Run\-Type}{p.}{class_dsp_run_type} class makes use of an internal template class and public template methods to allow users to get and set the contained variable as any type they wish. A \doxyref{Dsp\-Run\-Type}{p.}{class_dsp_run_type} object also has the ability to change type at any point during program execution. Built-\/in typecasting and error checking (via the Run\-Type\-Cast() method) prevents critical runtime errors from occurring when signal types are mismatched.
