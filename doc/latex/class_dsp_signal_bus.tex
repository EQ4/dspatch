A \doxyref{Dsp\-Signal\-Bus}{p.}{class_dsp_signal_bus} contains Dsp\-Signals (see \doxyref{Dsp\-Signal}{p.}{class_dsp_signal}). Via the Process\-\_\-() method, a \doxyref{Dsp\-Component}{p.}{class_dsp_component} receives signals into it's \char`\"{}inputs\char`\"{} \doxyref{Dsp\-Signal\-Bus}{p.}{class_dsp_signal_bus} and provides signals to it's \char`\"{}outputs\char`\"{} \doxyref{Dsp\-Signal\-Bus}{p.}{class_dsp_signal_bus}. Although Dsp\-Signals can be acquired from a \doxyref{Dsp\-Signal\-Bus}{p.}{class_dsp_signal_bus}, the \doxyref{Dsp\-Signal\-Bus}{p.}{class_dsp_signal_bus} class provides public getters and setters for manipulating it's internal \doxyref{Dsp\-Signal}{p.}{class_dsp_signal} values directly, abstracting the need to retrieve and interface with the contained \doxyref{Dsp\-Signal}{p.}{class_dsp_signal} objects. Likewise, signals can be added and removed from a \doxyref{Dsp\-Signal\-Bus}{p.}{class_dsp_signal_bus} via public methods.
